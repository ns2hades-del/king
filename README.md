# Site VITCH

Ce site web présente VITCH, une entreprise fintech fondée le 13 avril 2026 pour révolutionner les paiements numériques en Afrique.

## Pages

- `index.html` : Page d'accueil avec présentation, produits et fonctionnalités
- `services.html` : Détails des services offerts
- `about.html` : À propos de l'entreprise
- `contact.html` : Informations de contact et formulaire
- `dashboard.html` : Tableau de bord utilisateur (démo)
- `standard.html` : Détails de la carte Standard
- `premium.html` : Détails de la carte Premium
- `vip.html` : Détails de la carte VIP

## Utilisation

Ouvrez `index.html` dans votre navigateur web pour explorer le site.

## Technologies

- HTML5
- Tailwind CSS (via CDN)
- JavaScript (animations simples)

## Dépannage

- Assurez-vous que votre navigateur supporte HTML5 et JavaScript.
- Si les styles ne se chargent pas, vérifiez votre connexion internet pour Tailwind CSS.
